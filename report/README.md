# Report

This folder contains the LaTeX source and report figures for the written technical report.

- `source/main.tex`: main report source.
- `source/references.bib`: bibliography.
- `source/`: figures included by the report source.

The root-level `../technical_report.pdf` is the rendered public copy linked from the main README. The local `report/report.pdf` copy is ignored and only kept for convenience.
