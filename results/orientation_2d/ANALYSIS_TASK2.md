The initial two-dimensional tests require that the one-dimensional Toro Table 4.1 Riemann problems are embedded in two dimensions on a square domain, with the discontinuity first defined by the x-coordinate and then by the y-coordinate. In these two cases the solutions should remain consistent with the corresponding one-dimensional behaviour, so the field should vary primarily in the direction normal to the initial discontinuity and remain approximately constant in the transverse direction. The brief also requires that the tests are repeated with and without adaptive mesh refinement (AMR) using comparable effective resolution, and that when AMR is used the mesh or mesh patches are shown clearly without obscuring the data. Finally, the discontinuity must be rotated so it is not perpendicular to either coordinate axis, which checks that the two-dimensional flux implementation behaves correctly for genuinely oblique wave propagation. Our Task 2 dataset satisfies these requirements using uniform runs at 400×400 and AMR runs with base 200×200 and one refinement level (max_level=1), which matches the finest effective spacing between the two modes while keeping the 2D study at a single sufficiently accurate resolution as intended by the brief.

For the x-split configuration (left state for x<0.5 and right state for x>0.5), the expected 2D outcome is an effectively one-dimensional solution that depends on x but not on y. This behaviour is visible in the density field for the representative Toro test 1, where the wave structure appears as vertical bands across the domain in the uniform-grid result `results/orientation_2d/figures/main_text/test1/rho_t1_xsplit_uniform.pdf`. The corresponding AMR result `results/orientation_2d/figures/main_text/test1/rho_t1_xsplit_amr_with_patches.pdf` shows the same wave pattern while also displaying patch boundaries. These patches are thin enough to preserve readability, and refinement is localized to the sharp features, as required for a meaningful AMR demonstration.

For the y-split configuration (left state for y<0.5 and right state for y>0.5), the correct solution is the x-split result rotated by ninety degrees. This is observed in the representative Toro test 1 density field, where the uniform-grid y-split result `results/orientation_2d/figures/main_text/test1/rho_t1_ysplit_uniform.pdf` exhibits horizontal bands corresponding to the rotated wave structure, and the AMR y-split result `results/orientation_2d/figures/main_text/test1/rho_t1_ysplit_amr_with_patches.pdf` reproduces the same rotated behaviour with refinement again concentrated near steep gradients. This rotation equivalence is quantified in the Task 2 metric tables by comparing centerline profiles from the x-split and y-split runs. For Toro test 1, the AMR rotation-equivalence error in density is L1=2.439e-16 and L∞=2.331e-15, while the uniform-grid rotation-equivalence error is exactly zero to machine precision, as reported in `results/orientation_2d/checks/task2_metrics_by_test.md`. We extend this beyond the minimum requirement by repeating the same 2D embedding for Toro tests 2–5 and placing these results in the appendix figure set. In the nonzero-velocity cases, rotational consistency is particularly sensitive to correct interpretation of the Toro initial velocities. The final dataset includes a corrected y-split orientation in which Toro states are treated as normal and tangential velocities relative to the discontinuity, and then rotated into Cartesian components. With this correction applied, the rotation-equivalence errors remain at the level of floating point round-off across all tests, for example L1=1.061e-15 and L∞=7.772e-15 for Toro test 2 (AMR, density) and L1=1.268e-14 and L∞=3.730e-13 for Toro test 5 (AMR, density), again documented in `results/orientation_2d/checks/task2_metrics_by_test.md`.

The brief also asks that the 2D x-split and y-split solutions do not differ from the one-dimensional tests. A direct way to support this statement is to compare the 2D centerline profiles to the corresponding 1D reference solution. These comparisons are included in the metric tables for the main physical variables (rho, velocity, pressure, and specific internal energy). For Toro test 1, the centerline density error relative to the 1D exact solution is modest at the chosen resolution, and it is reduced when AMR is used at matched effective spacing. Specifically, the x-split centerline density error is L1=5.134e-03 (uniform) versus L1=3.977e-03 (AMR), with corresponding L∞ values 1.315e-01 (uniform) versus 8.093e-02 (AMR), as listed in `results/orientation_2d/checks/task2_metrics_by_test.md`. These checks are not intended as a full 2D convergence study, which the brief does not require, but they provide quantitative support for the expected 1D-consistent behaviour in 2D.

The final Task 2 requirement is an oblique discontinuity that is not perpendicular to either axis, which exercises the genuinely two-dimensional flux update. We use a diagonal split through (0.5,0.5) with normal direction proportional to (1,1), so the initial interface is aligned with the line x+y=1. In the representative Toro test 1 density field, the uniform-grid diagonal result `results/orientation_2d/figures/main_text/test1/rho_t1_diag_uniform.pdf` shows wave structures that remain oblique, rather than collapsing onto purely vertical or horizontal features, which is the qualitative signature of correct multi-dimensional propagation. The AMR diagonal plot `results/orientation_2d/figures/main_text/test1/rho_t1_diag_amr_with_patches.pdf` shows that refinement follows the oblique high-gradient regions while leaving large smooth regions unrefined. Some mild grid-alignment diffusion is expected on a Cartesian mesh when the discontinuity is not aligned with grid lines, and this is reflected in the diagonal symmetry metrics that compare x- and y-centerline density profiles for the diagonal case. For Toro test 1, these values are L1=8.468e-04 and L∞=8.296e-03 on the uniform grid and L1=1.076e-03 and L∞=4.786e-03 with AMR, as recorded in `results/orientation_2d/checks/task2_metrics_by_test.md`. These magnitudes are consistent with the intended interpretation of the diagonal test as a verification of 2D flux behaviour rather than a demand for perfect discrete symmetry.

Across all Toro tests 1–5, we record per-test density normalization bounds in `results/orientation_2d/checks/rho_extrema_by_test.tsv`, which enables fair within-test comparisons between uniform and AMR visualizations. The final outcome of Task 2 is therefore a consistent set of 2D verification results that demonstrate correct embedding of the 1D problems in 2D, correct rotation behaviour between x- and y-defined discontinuities, correct visualization and localisation of AMR patches, and correct qualitative behaviour for an oblique discontinuity that probes the true two-dimensional flux path. The inclusion of the full Toro test set in the appendix strengthens this conclusion without changing the core reporting approach, since the brief explicitly allows a single sufficiently accurate 2D resolution and does not require repeating the full convergence analysis in two dimensions.
