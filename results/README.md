# Results

This folder keeps the compact, frozen result artifacts used by the public report and README.

- `riemann_1d/`: one-dimensional validation figures, error tables, and AMR coverage summaries.
- `orientation_2d/`: two-dimensional orientation and diagonal-split figures and metrics.
- `quadrant_scaling/`: AMR/MPI timing figures, speedup tables, sensitivity summaries, and full-field accuracy diagnostics.

Large raw plotfiles and rerun logs are intentionally not tracked. They can be regenerated with the workflows in `../experiments/`.
