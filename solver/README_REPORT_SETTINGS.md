This code pack is organised around task-specific runner scripts.

The report settings are reproduced by the scripts under `task1/scripts`, `task2/scripts`, and `task3/scripts`. Base input templates under `task1/inputs`, `task2/inputs`, and `task3/inputs` are included for reference only and may be overridden by those scripts at run time.

The task scripts and analysis utilities expect the AMReX plotting tools under `AMREX_HOME/Tools/Plotfile/`.
