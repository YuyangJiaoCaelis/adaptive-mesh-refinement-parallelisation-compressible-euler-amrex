The experiment workflows are organised around the three report studies.

The report settings are reproduced by the scripts under `riemann_1d/scripts`, `orientation_2d/scripts`, and `quadrant_scaling/scripts`. Base input templates under each experiment's `inputs/` directory are included for reference only and may be overridden by those scripts at run time.

The experiment scripts and analysis utilities expect the AMReX plotting tools under `AMREX_HOME/Tools/Plotfile/`.
